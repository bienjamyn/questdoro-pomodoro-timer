## Download the app

**[Download QuestDoro V1.1.1](../../releases/latest)** (macOS 14+)

## Install

1. Download `QuestDoro.zip` from the link above
2. Unzip the file
3. Drag `QuestDoro.app` to your Applications folder or Desktop
4. Open QuestDoro and time to focus!

> **Note:** On first launch, your Mac may block the app. Go to **System Settings → Privacy & Security** and click "Open Anyway".

## Features

- Pixel art aesthetic
- Lives in the menu bar
- Focus and break timer modes


## Controls

![QuestDoro Controls](QuestDoro%20Controls.gif)

- Hover to expand, click to control
- Click the restart button twice to reset the entire session
- Double-click the timer to enter a custom time (press Enter to save / Esc to discard changes)
